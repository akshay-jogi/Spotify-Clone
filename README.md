
# Spotify Clone




## 🎯 Aim
I wanted to build a project using what I learned in HTML and CSS.
I haven't used JavaScript yet because I haven't learned it yet.

## Screenshots

![image alt](https://github.com/akshay-jogi/Spotify-Clone/blob/b8194b97ee52c27ba30849c017425b32d6332df2/Spotify%20Clone.PNG)

## 🔍 What I Focused On
- 🏗️ **Structure** — Clean, semantic HTML from scratch
- 📐 **Placement** — Every element placed exactly where it belongs
- 📱 **Responsiveness** — Layout that works across all screen sizes
- 🎨 **Theme** — Spotify's dark, minimal look using pure CSS

---

## 💡 What I Learned
- 🔗 How to use **CDN links** in HTML (Google Fonts & Font Awesome)
- 🎨 Styling scrollbars using **`::-webkit-scrollbar`** and its thumb
- 📦 How to use **Flexbox** and its properties
